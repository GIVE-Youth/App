# App
Mobile app for GIVE Youth Organization

# Screens - Bottom Navigation Bar in Order (Left to Right):
1. Photomath
- ability to sumbit pictures of math problems, will solve for you
- will be able to contact tutors through chat application

2. Map
- Map of all tutoring centers
- Each pin on map is linked to page describing center (address, POC, CM, schedule)

3. Profile/Home
- Account login
- Tutor or Tutee
- Member of which center
- Hours log (for tutors)
- Tutor assignment/preference (for tutees)

4. About/Contact US
- Executives and Board of Directors Info
- Mission Statement, Purpose of GIVE

5. Settings
- Customize notifications
- Change account settings
